# Object-Oriented Programming Final Project

The final project asks students to implement a practical object-oriented system
related to their future career or personal technical interests. The system should
not be only a theoretical explanation. It must include executable source code,
class design, and a report explaining how object-oriented programming concepts
were applied.

The report should include an introduction, related work, proposed system,
implementation and usage results, limitations, and conclusion. Screenshots of
the running program or important command-line outputs can be included as
evidence that the system works.

The project is evaluated based on whether the implementation clearly uses
encapsulation, abstraction, inheritance, and polymorphism. A good project also
explains why the chosen architecture is useful for the selected domain.
